# Smart Leads Dashboard

A full stack lead management dashboard built using:
- React + TypeScript
- Node.js + Express
- MongoDB

## Features
- Login system
- Add leads
- Leads dashboard
- Stats cards

## Run Locally

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev
